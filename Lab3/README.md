## Lab3 — Налаштування вікторини та діалог завершення гри

Ця лабораторна робота продовжує єдиний проєкт вікторини з Lab1–Lab2.

### Завдання

Додати логіку налаштування гри (рівні складності, кількість елементів, швидкість і тд) за допомогою `localStorage`, локального стейту чи контексту. Обов'язковою умовою є використання нескладної форми з `yup`/`formik`/`react-hook-form` чи аналогів. Також необхідно використати портали для створення діалогового вікна про завершення гри. Вікно включає інформацію про завершення гри і можливість розпочати наступний тур чи цей заново.

### Реалізація

- **Налаштування гри**:
  - Контекст налаштувань (`SettingsContext`) з збереженням у `localStorage`.
  - Форма налаштувань на стартовій сторінці на базі `react-hook-form` + `yup`:
    - Тема (категорія) питань.
    - Складність питань.
    - Кількість запитань.
    - Час на одне запитання (секунди).

- **Розширена логіка**:
  - Хук `useQuizEngine` враховує налаштування (тема, складність, кількість запитань).
  - Фільтрація та обмеження питань відповідно до налаштувань.

- **Портали**:
  - Діалогове вікно завершення гри (`GameOverDialog`) створено через `ReactDOM.createPortal`.
  - Вікно відображає підсумок гри та пропонує опції:
    - Розпочати новий тур.
    - Розпочати цей тур заново.

### Запуск

```bash
npm install
npm run dev
```
