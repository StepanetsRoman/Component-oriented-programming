## Lab4 — Роутинг та стилізація вікторини

Ця лабораторна робота продовжує єдиний проєкт вікторини з Lab1–Lab3.

### Завдання

Необхідно створити роутинг між сторінками. Роутинг з користувачем має бути динамічним (з використанням id користувача). Необхідно додати стилізацію нашого застосунку. Будь-яким із методів `styled-components`, `Tailwind`, `Css модулі`.

### Реалізація

- **Роутинг**:
  - Роутинг на базі `react-router-dom` (v6) з **динамічним id користувача** в URL (`/user/:userId/...`).
  - Сторінка вибору користувача `UserSelectPage` (`/`).
  - Всі маршрути прив'язані до конкретного користувача.

- **Стилізація**:
  - Стилізація ключових елементів через **`styled-components`**:
    - `Layout` — основний лейаут застосунку.
    - `QuizHeader` — заголовок з градієнтним текстом.
  - Окремий локальний `globals.css` (без прив'язки до зовнішніх шляхів).

- **Структура маршрутів**:
  - `/` — вибір `userId` (форма переходу до вікторини).
  - `/user/:userId/start` — налаштування вікторини (форма з `react-hook-form` + `yup`).
  - `/user/:userId/game` — основна сторінка гри.
  - `/user/:userId/results` — сторінка результатів для конкретного користувача.
  - `/user/:userId/history` — історія ігор для конкретного користувача.

### Запуск

```bash
npm install
npm run dev
```
