## Каркас гри «Вікторина (Trivia Quiz)»

Цей проєкт — це **каркас React-застосунку** для гри-вікторини без бізнес-логіки та без роутингу.

### Структура

- **Сторінки**:
  - `StartPage` — стартова сторінка з налаштуваннями (поки плейсхолдер).
  - `GamePage` — основна сторінка гри з запитанням, варіантами відповідей та панеллю результатів.
  - `ResultsPage` — сторінка підсумкових результатів.
- **Компоненти**:
  - `Layout` — спільний лейаут для всіх сторінок.
  - `QuizHeader` — верхній блок із назвою гри.
  - `QuestionCard`, `AnswerList`, `Scoreboard` — структурні компоненти-плейсхолдери.

Усі кнопки та селекти **не мають логіки** (відключені), головний акцент — на **правильному поділі на компоненти** та базовому сучасному UI.

### Запуск

```bash
npm install
npm run dev
```
