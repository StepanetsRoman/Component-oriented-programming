## Lab2 — Вікторина з логікою на хуках

Цей проєкт продовжує каркас з Lab1 та додає **основну логіку** за допомогою **React-хуків**.

### Завдання

Додавання основної логіки за допомогою використання хуків. Додати необхідний стан для кожної сторінки, а також додати логіку програми за допомогою кастомних хуків. Компонент повинен залишатись максимально чистим.

### Реалізація

- **Кастомні хуки**:
  - `useQuizEngine` — містить всю логіку гри (запитання, відповіді, підрахунок очок, серії правильних відповідей).
  - `useQuizNavigation` — керує перемиканням між сторінками (START, GAME, RESULTS, HISTORY).

- **Компоненти**:
  - Компоненти сторінок (`StartPage`, `GamePage`, `ResultsPage`, `HistoryPage`) залишаються максимально «чистими».
  - Вся бізнес-логіка винесена в кастомні хуки.

- **Функціональність**:
  - Відображення питань з кількома варіантами відповідей.
  - Підрахунок очок та серій правильних відповідей.
  - Збереження результатів гри в `localStorage`.
  - Перегляд історії попередніх ігор.

### Запуск

```bash
npm install
npm run dev
```
